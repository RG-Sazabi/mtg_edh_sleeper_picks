# Memory Index

This file is the index for project-level persistent memory.
Claude Code reads this automatically to recall prior context across sessions.
Each entry points to a file in this directory with the full memory content.

Keep entries under ~150 characters each. The index is loaded in full on every session.

- [References](references.md) — Pointers to external systems (GitHub repos, dashboards, docs)
